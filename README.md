## Summary
 - MJ Flux is an Ecommerce website for shopping clothes  online

## Technology Used
- HTML
- CSS
- Javascipt
- React JS
- Next JS
- Sanity

## Features
- Add to Cart
- Delete from Cart
- Multiple item views
- Similar products view
- Stripe Payment

## Installation
- git clone URL (git@github.com:miriamnaki/MJ-Flux.git)
- cd into project
- cd into myeshop and  npm run dev
- In another terminal cd into sanity_myeshop and run sanity start
